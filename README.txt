Entrega intermedia del curso de Python brindado por CoderHouse

En este proyecto se busc√≥ la creacion de un servidor pensado para registros de datos en una biblioteca para ingresar libros, afiliados y registrar retiros de libros. 

Al ingresar al local host, tenemos que insertar /AppBiblioteca/ en la url para ingresar al inicio del servidor para que luego permita una navegacion entre diferentes vistas, en las cuales encontramos un formulario para completar y que lo guarda en una base de datos.
Dentro de esas vistas tenemos:
- inicio/, en donde hay una presentacion y se observan botones que redireccionan a otras vistas.
- afiliados/, la cual presenta un formulario para ingresar afiliados con su nombre, apellido y email.
- retiros/, donde encontramos otro formulario para registrar retiros completando con un nombre, el codigo del libro a retirar y la fecha de retiro
- libros/ la cual tambien contiene un formulario con un titulo de libro a ingresar, el autor de dicho titulo, un tipo refiriendose a la clasificacion del libro y un codigo numerico. 
- Una linea de texto que nos dirige a la busqueda de un libro por medio del codigo del libro.
